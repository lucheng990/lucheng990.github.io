> What's Machine Learning?

目的：从Data里面找到一组function 


### 框架
1. 定A set of function (Model *f1*,*f2*···)
2. 确定Goodness of function *f*
3. Training Data
4. 通过训练训练集来改善*f*,pick the best function*f**
5. 使用*f** 来应用
